##  About The Project

**Aurora** is an AI-powered poem generator that creates beautiful, customized poetry based on themes, moods, and user prompts. Built using advanced prompt engineering techniques and Generative AI, Aurora delivers unique poetic masterpieces every time.

Simply choose a theme, optionally add your own custom prompt, and let Aurora's AI craft a beautiful poem just for you!

| Feature | Description |

##  How To Use

1. Visit the **[Live Demo](https://v0-aurorapoemgenerator-he4marau3-lekhavsb-7380s-projects.vercel.app/)**
2. Choose a **theme** from:
   - 🌿 Nature | 🌌 Cosmic | 💫 Love | 🌙 Mystery | 🌊 Ocean
3. *(Optional)* Add a **Custom Prompt** for personalized poems
4. Explore **Advanced Generation Options** for more control
5. Click **"Generate Poem"** and enjoy your masterpiece! ✨

##  Run Locally

```bash
# Clone the repository
git clone https://github.com/lekhashree21/Aurora-Poem-Generator.git
cd Aurora-Poem-Generator

# Install dependencies
npm install

# Run development server
npm run dev

# Open browser at
http://localhost:3000
```
